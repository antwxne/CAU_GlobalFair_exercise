## Build

If problems with CMake version, upgrade yours or change in [CMakeList.txt](CMakeLists.txt).

### CLI

```shell
mkdir -p build
cd build
cmake .. && cmake ---build .
```

### Code editor / IDE

Also it's possible to run the code from VSCode, Clion, VSC. 
Just load the CMakelist then run the exercise.
